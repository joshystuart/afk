# Roadmap: AFK (Away From Keyboard)

## Overview

AFK’s **v1 program** extends the secure container-based workflow into a daily-driver platform. **v0.3.4** shipped Session UX, Skills provisioning, and the Workspace API with file explorer and @-context. Remaining v1 work: diff review, git automation, and a pluggable multi-agent runner.

## Milestones

- ✅ **v0.3.4 Workspace release** — Phases 1–3 (Session UX, Skills, Workspace API & Explorer) — shipped 2026-04-18 — [full roadmap archive](milestones/v0.3.4-ROADMAP.md)
- 🚧 **v1 remainder** — Diff pipeline, git automation, agent runner (planned)
