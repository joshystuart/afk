Terminal unread detection — listen for `terminal.data` socket events when chat tab is active:
```typescript
const { socket } = useWebSocket();
React.useEffect(() => {
  if (!socket || !id) return;
  const onTerminalData = (payload: { sessionId: string }) => {
    if (payload.sessionId !== id) return;
    if (activeTab !== 'terminal') {
      setTerminalUnread(true);
    }
  };
  socket.on('terminal.data', onTerminalData);
  return () => { socket.off('terminal.data', onTerminalData); };
}, [socket, id, activeTab]);
```

Chat unread detection — listen for `chat.stream` socket events when terminal tab is active:
```typescript
React.useEffect(() => {
  if (!socket || !id) return;

      setChatUnread(true);
    }
  };
  socket.on('chat.stream', onChatStream);
  return () => { socket.off('chat.stream', onChatStream); };
}, [socket, id, activeTab]);
```
