## Trust Boundaries

| Boundary      | Description                                                        |
| ------------- | ------------------------------------------------------------------ |
| Browser → API | User-provided skills directory path sent via UpdateSettingsRequest |
| Browser → API | User-provided mountSkills boolean sent via CreateSessionRequest    |

## STRIDE Threat Register

| Threat ID | Category  | Component                        | Disposition | Mitigation Plan                                                                                                                                                |
| --------- | --------- | -------------------------------- | ----------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| T-02-07   | Tampering | UpdateSettingsRequest (web)      | mitigate    | Server-side validation via MountPathValidator (Plan 01 T-02-01). Frontend sends raw string; server rejects forbidden paths. No client-side bypass possible.    |
| T-02-08   | Spoofing  | CreateSessionRequest.mountSkills | accept      | Boolean field — only affects whether skills are mounted for the user's own session. No cross-user impact. Server validates via class-validator `@IsBoolean()`. |

</threat_model>
